/***********************************************************************************
Function   : Enter、Leave	
           : 信号量控制，保证通信为同步方式，与Send和Receive配合使用
************************************************************************************/
		
void Enter( BYTE ComID );

实例:

BYTE byArray[10] = {0x00, ….};
Enter(PLC);                         // 申请串口资源
Send(PLC, byArray, 10);
Receive(PLC, byArray, 10, 0, 6);
Leave(PLC);                         // 释放资源
