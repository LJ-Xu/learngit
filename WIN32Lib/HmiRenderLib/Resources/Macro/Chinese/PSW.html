其类型为 unsigned short（即WORD）

实例:

	
PSW[300]++;                               // 将PSW[300]按字++
DWORD dwValue = *(DWORD*)(PSW + 300);     // 将PSW[300]、PSW[301]里面的值赋值给一个双字
float fValue = *(float*)(PSW + 300);      // 将PSW[300]、PSW[301]里面的值按浮点数格式读出
*(DWORD*)(PSW + 300) = dwValue;           // 将一个双字赋给PSW[300]、PSW[301]
