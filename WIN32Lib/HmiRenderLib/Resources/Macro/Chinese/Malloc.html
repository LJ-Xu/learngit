/***********************************************************************************
Function    : Malloc替代标准库函数中的malloc使用。功能完全相同
Note        : 申请的空间请及时释放
      Size  : 申请空间大小（bytes）
      Return: 被申请空间的地址，等于NULL表示申请失败
************************************************************************************/
		
void *Malloc( UINT size )

实例:

BYTE* pBuffer = Malloc(10);
Free(pBuffer);
